# End-to-End Business Intelligence Dashboard

## Project Overview

This project is an End-to-End Business Intelligence Dashboard developed using Python. It transforms raw business data into meaningful insights through data analysis and visualization.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Features

* Data Loading and Cleaning
* Data Merging
* KPI Analysis
* Sales and Profit Visualization
* City-wise Analysis
* Category-wise Pie Chart
* Business Insights Dashboard

## Results

* Total Sales: 36000
* Total Profit: 6000
* Total Customers: 5
* Total Orders: 10

This project demonstrates an end-to-end Business Intelligence workflow for analytics and decision-making.
